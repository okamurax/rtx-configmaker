# RTX ConfigMaker

ヤマハ **RTX1220** のコンフィグを GUI からリアルタイム生成する macOS ネイティブアプリ（SwiftUI）。
左のパネルでチェック・スライダー・入力を操作すると、右側に設定コマンドが即座に生成されます。保存機能は無く、コピペで実機へ投入する用途を想定しています。

## 特長

- **リアルタイム生成** — 操作に応じて右のコンフィグが即時更新
- **機能カテゴリ単位のON/OFF** — 各カテゴリのチェックを外すと、その範囲を出力から除外（部分スニペット作成に便利）
- **4カラム等幅レイアウト** — 入力 / 生成コンフィグ / 既存コンフィグ貼り付け / 差分
- **差分表示** — 貼り付けた既存コンフィグと生成結果を行単位で比較（「生成だけ」「既存だけ」の2分類、前後空白・改行コード差は無視）

### 対応する設定項目

- 基本設定（文字コード、ログインタイムアウト、ログインユーザー、パスワード案内）
- LAN / WAN（PPPoE / DHCP / 固定IP / IPoE〔DS-Lite・MAP-E〕）
- DHCP サーバー / DNS / NAT・IPマスカレード / ポート開放（静的NAT）
- 拠点間VPN（IPsec：固定IP相互・センター側・拠点側）
- リモートアクセスVPN（L2TP/IPsec）
- 標準セキュリティフィルタ / 管理アクセス / NTP / Syslog

## 動作環境

- macOS 13 以降
- Swift 5.9 以降（ビルド時）

## ビルドと実行

```sh
# そのまま実行
swift run

# ダブルクリック可能な .app を生成
./build_app.sh
open "RTX ConfigMaker.app"
```

## 注意

- 生成されるコンフィグは RTX1220 の一般的な構成に基づく **雛形** です。**実機投入前に必ず内容を確認**し、ご利用の回線・ファームウェア世代・契約サービスに合わせて調整してください。
- IPoE（v6プラス / OCNバーチャルコネクト / transix 等）の一部コマンドはファームウェア世代により表記が異なる場合があります。
- 機器のパスワード（`administrator password` 等）は仕様上コンソールでの対話設定が必要なため、コンフィグにはコメントで案内を出力します。
